# TodoReminder
Android (Kotlin + Jetpack Compose): To-Do list z przypomnieniami (AlarmManager + notyfikacje) + Room + DataStore.

## Import do Android Studio
1. Pobierz i rozpakuj ZIP
2. Android Studio → File → Open → wskaż folder `TodoReminder`
3. Gradle Sync

## Notyfikacje
Na Android 13+ aplikacja poprosi o zgodę na powiadomienia.

## Wrapper
gradle-wrapper.jar: nie znaleziony w środowisku generowania; Android Studio i tak zaimportuje projekt, ale CLI wrapper może wymagać wygenerowania.
